# Rochabot - Auto AFK Bot for Aternos (Render Hosting)

✅ Auto reconnect  
✅ Respawn if dies  
✅ Keep Render alive  
✅ Sends messages on join  
✅ Periodic wake-up for Aternos  

### To deploy:
1️⃣ Upload this folder to GitHub  
2️⃣ Connect repo to Render → Web Service  
3️⃣ Build command: `npm install`  
4️⃣ Start command: `npm start`  
5️⃣ Done ✅
